Assignment 3: JavaScript

Due February 11 at 6pm

We talked more about JavaScript this week, and we'll exercise some of our new knowledge around it while building upon HTML and CSS.

Resources

The JavaScript resources suggested last week still apply:

  • The Khan Academy Intro to JavaScript online course is popular
  • Codecademy's Introduction to JavaScript is also worthwhile
  • NodeSchool has a series of exercises written in Glitch projects called JavaScripting
  • If you're looking for a resource on JavaScript that you can read through, the first few chapters of Eloquent JavaScript is a good source. It goes a little more in depth than you need to right now, so feel free to skip and skim parts of it.
  • Another readable resource is javascript.info
  • Learn JavaScript
  • JavaScript for Cats

Focus on the core concepts of JavaScript: variables, arrays, objects, and functions.

Create a map using more advanced JavaScript and Leaflet features

This week we covered advanced JavaScript and Leaflet features such as popups, feature styling, filtering, and events. This assignment is an opportunity to apply each of these to either your map from last time or a new one.

  1. Decide whether you are creating a new map for this assignment or extending your map from the last assignment. Either way, create a new Glitch app for this assignment—you can remix your app from the last assignment.
  2. Add popups to the map that provide more information about the features.
  3. Style the popups with CSS.
  4. Style the features.
  5. Use event listeners to respond to an event of your choosing and change the map in some way. For example, this could be a link or button on your page that filters the data on the map when it is clicked. It could also be some HTML form input that filters the data in a more complex way, or an event that is more specific to Leaflet such as a click on a feature.

Final project proposal

Flesh out your thoughts for one of your project ideas from last time. See the final project assignment for a refresher on what is required. Write about:

  1. The project idea: what is it? why do you want to explore it?
  2. Data sources you will likely use.
  3. What features your project will include.
  4. What you want the project to look like. While not required, a rough sketch of what you want the site to look like is strongly recommended.
  5. Your timeline for the project. Remember that it is due the last night of class, October 16. Be ambitious but realistic—what are the phases of the project? What is the bare minimum you could accomplish for the class and while being happy with it? Which parts would you consider continuing to work on after class?

Submit a (1/2- to 1-page) PDF or Word document in the upload field set aside for it on the assignment submission form. We will briefly and informally talk about these at the beginning of class on the evening this assignment is due, so please be prepared for that.
