Resources

CSS

  • Codecademy has a course on CSS: Learn CSS (The first five sections, don't bother with the pro lessons).
  • MarkSheet has a good free HTML and CSS tutorial.
  • CSS-Tricks has good beginner-to-intermediate tutorials, and you might appreciate HTML & CSS -- The VERY Basics.
  • The first three sections on this HTML Dog page list the more common CSS properties you'll be using.
  • This CSS cheat sheet is a terse but you might find it useful.
  • There are more exhaustive references on the Mozilla Developer Network.

JavaScript

  • Codecademy's Introduction to JavaScript is also worthwhile
  • The Khan Academy Intro to JavaScript online course is popular
  • NodeSchool has a series of exercises written in Glitch projects called JavaScripting
  • If you're looking for a resource on JavaScript that you can read through, the first few chapters of Eloquent JavaScript is a good source. It goes a little more in depth than you need to right now, so feel free to skip and skim parts of it.
  • Another readable resource is javascript.info
  • Learn JavaScript
  • JavaScript for Cats

Create a page with a Leaflet map on it

Let's create an webpage from scratch using HTML and CSS, then add a Leaflet map to it using JavaScript.

  1. Remix this Glitch site into your account. For your convenience, it contains a dataset you can use to create a map. It contains some of the world's airports, via Natural Earth Data.
  2. Load Leaflet and add a map to the page. Use the example Glitch site we have been working with to help.
  3. Add the data that is included in the site's assets folder (airports.geojson).
  4. Edit the HTML and add some content to it.
  5. Edit the CSS file to style it. Consider fonts, colors, and positions when making your changes. Use at least three selectors, and select based on your elements' classes for at least one. The styles should be in a CSS file linked in the <head> of your document, like this: 
    <!doctype html>
<html>
    <head>
        <link rel="stylesheet" href="/styles.css">
    </head>
    <body>
    </body>
</html>
  6. Submit a link to the public page on Glitch (not the link you use for editing, the one you get when you click Show Live).
