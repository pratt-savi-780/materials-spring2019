Events, APIs, and intermediate Leaflet

Make a page with a Leaflet map. The page must:

  • use event listeners in some way (such as responding to map features or a button or other element being clicked)
  • style features on the map based on the features' attributes using an if statement

the page may:

  • use the NYC open data portal and the SODA API

You are welcome to work on finding data and making preliminary maps toward your final project for this assignment.
