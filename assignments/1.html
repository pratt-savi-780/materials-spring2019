Assignment 1: HTML

Due January 28 at 6pm

We talked about HTML this week, and we'll exercise some of our new knowledge around it by using HTML to make our own pages.

Resources

  • My HTML Reference should get you started.
  • Codecademy has a course: Learn HTML (Just do the first section, "Elements and Structure", don't bother with the pro lessons).
  • MarkSheet has a good free HTML tutorial.
  • If you're looking for a resource on HTML that you can read through, I'd recommend the first three chapters of HTML & CSS is Hard. For now I would stick with editing code in Glitch, so you can skip the parts that mention a text editor named Atom.

Create a page with HTML

Let's create an webpage from scratch using HTML. Pick something you're interested in and make a one-page site in Glitch about that topic.

We have not yet covered CSS, which will add style to an HTML page. Please do not spend time styling your page for this assignment. Stick to HTML for now.

  1. Create a site in Glitch.
  2. Edit the HTML page (index.html):
    • add some text
    • add at least one image (upload it to Glitch first)
    • add at least one link
    • experiment with some other HTML tags as outlined in the HTML Reference
  3. Submit a link to the public page on Glitch (not the link you use for editing).
