In-class Exercise: Recap, Events

Part 1: Leaflet recap

Please work with a partner on the following exercise, in which you will make a Leaflet map using a number of features we have covered so far in class.

  1. Create a new Glitch site (New Project > hello-webpage). You're free to use any Glitch examples or previous work you have done as a guide, but please do start from a fresh site.
  2. Add a Leaflet map to the site.
  3. Style the map so that it takes up some of the page but leaves room for some explanatory text.
  4. Use a base map of your choosing.
  5. Using the API, load data from either:
    • 311 service requests or
    • NYPD vehicle collisions
    Either way, use API parameters to filter the data in some manner.
  6. Style the data. Using if and else, style some features differently than others.
  7. Add popups using at least two attributes from the features.

Part 2: A zoom out button

  1. Remix this Glitch site in your account.
  2. The zoom in button should work on the resulting page.
  3. Implement JavaScript for the zoom out button—it should be very similar to the code for the zoom in button.

Part 3: Filtering data on an event

  1. Remix this Glitch project into your Glitch account.
  2. Confirm that the Brooklyn button works.
  3. Implement the Queens button—make it so that when the Queens button is clicked, data is loaded for Queens. Follow the example of the Brooklyn button in the code to select the button and make it do what you would like.

Part 4: Dropdown events

  1. Remix this Glitch project into your Glitch account.
  2. Confirm that the dropdown works for Brooklyn.
  3. Change the HTML for the dropdown (<select>) to add the other boroughs.

Part 5: Listening for Leaflet events

  1. Remix this Glitch project into your Glitch account.
  2. In the JavaScript, listen for more map events and log out the name of the event. For example, copy 
    map.on('moveend', function () {
  console.log('moveend');
});
    paste those lines (so the moveend listener still exists) and change moveend in the new listener, like this: 
    map.on('click', function () {
  console.log('click');
});
    Don't forget to change what is logged out, too, so you can see which event happened.
  3. Do the above process until you have five different map events. Pay special attention to:
    • click
    • dblclick
    • mousedown
    • mouseup
    • move
    How do the above events differ from each other?
